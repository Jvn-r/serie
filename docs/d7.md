# DELI 7 - Process Lifecycle framework

## Objective
- begin implementing process lifecycle tracking and snapshot system for lifecycle

## Learnings
- `atoi()` can directly convert PID strings from `readdir()` into integers
- process lifecycle detection needs comparing snapshots, not streaming events
- birth detection can be done from new entries
- But death detection requires a complete previous and current snapshot
- storing PIDs in arrays keeps the implementation simple and allows later optimization

## Outcomes
- created `pid_list` structure for process snapshots
- implemented PID collection from `/proc`
- dynamically resize PID arrays as process counts grow
- converted PID directory names into integer values
- established:
  - previous snapshot (`prev`)
  - current snapshot (`curr`)
- created framework for lifecycle comparison through `exists(prev, curr)`
- implement `exists()`
- detect process births
- detect process deaths
- begin process lifecycle tracking
